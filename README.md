# What is this project about?

This repository contains the work for the fourth assignment of the Advanced Simulation course of TU Delft by EPA133a Group 09

## Goals of this project

This assignment focuses on a graph analysis of the whole main road network fo N, R and Z roads in bangladesh. We study the probability of each of the edges of the graph shutting down, and the impact it would cause in terms of delays of goods transported by trucks.

## Files

- the ```/road_intersection/``` folder contains the cleaned dataset from the teachers and the code preparing it to be fed into the simulation. More information in the ```/road_intersection/README.md``` file
- the ```/model/``` folder contains the code relative to the graph analysis. More information in the ```/model/README.md``` file
- the ```/report/``` folder contains the report

## Dependencies 

- Pandas
- Geopandas
- Numpy
- Openpyxl

## How to run ?

Simply execute ```$ python /model/main.py``` and follow the instructions in the console. More information about the code can be found in the README in each folder.
